# Getting Started

You will need two terminals

terminal 1

```
npm install
./build.sh
cd packages/eth-atl-billing-contracts
npm run serve
```

in the console find your seed and import it to meta mask

terminal 2

```
cd packages/eth-atl-billing-contracts
npm run deploy
cd ../packages/eth-atl-billing-frontend-react
./start.sh
```
